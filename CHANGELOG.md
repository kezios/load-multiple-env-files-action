# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release
- Load multiple `.env` files in order
- Skip missing files by default
- Optional `fail-on-missing` parameter
- Optional `log-variables` parameter for debugging
- Support for comments and empty lines
- Variable override support (later files override earlier ones)
- Comprehensive test suite
- Detailed documentation and examples

### Features
- ✅ Multi-file loading
- ✅ Cascading configuration
- ✅ Missing file handling
- ✅ Variable name logging
- ✅ Clean summary output
- ✅ Conditional file support

## [1.0.0] - 2025-11-24

### Added
- First stable release
- Core functionality for loading multiple `.env` files
- Complete documentation
- Test coverage
